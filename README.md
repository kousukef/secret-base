# README

大学内オンラインフリマショッピングストア

機能一覧(使用技術)
* 商品の閲覧、出品、編集、削除、購入機能
* adminによる全ての商品の編集、削除機能
* 購入前、購入後のメッセージでの質問機能（ActionCable）
* ページネーション機能(kaminari)
* 商品検索機能(ransack)
* 認証機能(device)
* 画像添付機能(carrierwave)
* 多階層カテゴリ(ancestry)
